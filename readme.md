# Go Build

**Go Build** is a beginner friendly learning series where Go is learned by building real projects from scratch.  
The goal of this repository is to document a practical path from simple CLI tools to real web APIs using Gin.

This repo grows alongside the YouTube series and contains the source code for each episode.

## What you will learn
- Go fundamentals through hands on projects
- Building CLI applications
- Working with files and the OS
- Structuring Go projects
- Building HTTP servers
- Creating REST APIs with Gin

## Who this is for
Go Build is for:
- Beginners learning Go
- Developers coming from other languages
- Anyone who prefers learning by building instead of pure theory

## Projects in this repository
Each folder represents a project or episode from the series, starting simple and increasing in complexity.

Examples include:
- CLI tools
- File based applications
- Local utilities
- Web APIs built with Gin

## Getting started

### Prerequisites
- Go installed (1.20 or newer recommended)

### Run a project
Navigate into a project folder and run:
```bash
go run main.go
```
